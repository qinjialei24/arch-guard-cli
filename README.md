# x-cli 快速生成符合项目风格的代码，类似 angular-cli

## 使用
```bash
npm install x-cli -g
```

## 常用命令(必须在项目的**src**目录下运行)
- 创建页面：`xc g page Demo`
- 创建基础组件：`xc g c -ba (xc generate component --basic)`
- 创建业务组件：`xc g c -bu (xc generate component --business)`

## 常用缩写
- `generate => g`
- `component => c` 
- `--basic => -ba`
- `--business => -bu`
